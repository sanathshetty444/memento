# Security Policy

## Reporting a Vulnerability

**Do not report security vulnerabilities through public GitHub issues.**

Please email [sanathshetty444@gmail.com](mailto:sanathshetty444@gmail.com) with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** within 72 hours
- **Status update:** within 7 days
- **Fix target:** within 30 days for critical issues

## Disclosure Policy

We follow coordinated disclosure — vulnerabilities are fixed before public announcement. Credit is given to reporters unless they prefer anonymity.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.3.x   | ✅        |
| < 0.3   | ❌        |
